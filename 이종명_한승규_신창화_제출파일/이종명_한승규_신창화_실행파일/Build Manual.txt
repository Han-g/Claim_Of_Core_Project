						[빌드 메뉴얼]

실행 방법 >> Windows 환경에서 "Executable File\Claim_Of_Core.exe" 를 실행해 빌드한다.

1. 클라이언트 실행 후 로그인 화면에서 테스트 계정으로 접속한다.
   - ID	  : admin
   - PW : admin

2. 로비에서 방을 생성하거나 기존 방에 입장한다.

3. 방 화면에서 사용할 플레이어 슬롯을 선택한다.

4. 선택한 슬롯에서 캐릭터 종류(Striker / Guardian / Manipulator)를 선택한다.
   캐릭터 선택 시 서버로 캐릭터 선택 요청과 Ready 요청이 함께 전송되어 준비 상태가 된다.

5. 모든 플레이어가 READY 상태가 되면 방장이 Start 버튼을 눌러 게임을 시작한다.
