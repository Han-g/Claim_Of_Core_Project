발췌해야 하는 코드 흐름
--------------------
IceChillSpawner는 그대로 재사용해도 된다. 핵심 흐름은 아래와 같다.

void AIceChillSpawner::Tick(float DeltaTime)
{
    Super::Tick(DeltaTime);

    InitializeGameState();
    UpdateSpawner(DeltaTime);
}

void AIceChillSpawner::UpdateSpawner(float DeltaTime)
{
    if (!CanSpawn())
    {
        SpawnAccumulator = 0.f;
        return;
    }

    SpawnAccumulator += DeltaTime;

    if (SpawnAccumulator >= NextSpawnTime)
    {
        SpawnAccumulator = 0.f;
        SpawnChillZones();
        NextSpawnTime = GetRandomSpawnInterval();
    }
}

void AIceChillSpawner::SpawnChillZones()
{
    if (!ChillZoneClass || !GetWorld())
    {
        return;
    }

    const int32 SpawnCount = GetRandomSpawnCount();
    for (int32 i = 0; i < SpawnCount; ++i)
    {
        FActorSpawnParameters SpawnParams;
        SpawnParams.Owner = this;
        SpawnParams.SpawnCollisionHandlingOverride =
            ESpawnActorCollisionHandlingMethod::AdjustIfPossibleButAlwaysSpawn;

        GetWorld()->SpawnActor<AIceChillZone>(
            ChillZoneClass,
            GetRandomSpawnLocation(),
            FRotator::ZeroRotator,
            SpawnParams
        );
    }
}

현재 프로젝트에서 수정할 점
------------------------
현재 AIceChillSpawner는 클라이언트 월드에서 랜덤으로 AIceChillZone을 직접 생성한다. 멀티플레이에서 모든 클라이언트가 같은 위치에 같은 빙결 구역을 보려면 최종적으로 서버가 위치를 정하고 브로드캐스트해야 한다.

임시 구현:
 - 현재 AIceChillSpawner를 그대로 사용한다.

단계 2. 빙결 구역이 캐릭터를 얼리는 방식
====================================

현재 프로젝트의 문제점
-------------------
현재 AIceChillZone::ApplyFreezeToCharacter(AMyCharacter* Character)는 CharacterMovement를 직접 DisableMovement() 했다가 타이머로 복구한다.

문제:
 - AMyCharacter에 bFrozen 상태가 남지 않는다.
 - 횃불 아이템이 Target->IsFrozen()으로 해동 대상을 찾을 수 없다.
 - 서버 이동 계산과 상태가 분리된다.

따라서 현재 AIceChillZone::ApplyFreezeToCharacter는 최종적으로 AMyCharacter::ApplyFreeze() 또는 서버 빙결 요청 호출로 바꿔야 한다.

클라이언트 임시 구현 코드
----------------------
수정 대상:
 - D:\FinalProject\MainClient_Codes\Claim_Of_Core\Source\Claim_Of_Core\Map\IceCave\IceChillZone.cpp

현재 ApplyFreezeToCharacter를 아래처럼 단순화한다.

void AIceChillZone::ApplyFreezeToCharacter(AMyCharacter* Character)
{
    if (!IsValid(Character) || Character->IsDead())
    {
        return;
    }

    Character->ApplyFreeze();
}

최종 서버 권한 구현에서는 위 함수에서 직접 ApplyFreeze를 호출하지 않고, 서버가 브로드캐스트한 빙결 상태 패킷을 받은 뒤 UNetworkInstance가 Character->ApplyFreeze()를 호출하게 하는 편이 좋다.

bool AMyCharacter::CanReceiveStatusEffect(ERecStatusEffectType InStatusEffect) const
{
    if (!IsStatusEffectImmune())
    {
        return true;
    }

    switch (InStatusEffect)
    {
    case ERecStatusEffectType::Slow:
    case ERecStatusEffectType::Stun:
    case ERecStatusEffectType::Knockback:
    case ERecStatusEffectType::Freeze:
        return false;
    default:
        return true;
    }
}

3-8. ApplyRoleStats에 bFrozen 처리 추가
-------------------------------------
현재 AMyCharacter::ApplyRoleStats()는 항상 RoleType 기반 MaxWalkSpeed를 설정한다. 함수 시작 부분의 movement 처리에서 bFrozen이면 속도를 0으로 유지해야 한다.

void AMyCharacter::ApplyRoleStats()
{
    if (UCharacterMovementComponent* MoveComp = GetCharacterMovement())
    {
        if (bFrozen)
        {
            MoveComp->StopMovementImmediately();
            MoveComp->MaxWalkSpeed = 0.f;
        }
        else
        {
            const float BaseRoleMult = GetRoleSpeedMultiplier(RoleType);
            const float SkillMult = GetRoleSkillSpeedMultiplier();
            MoveComp->MaxWalkSpeed = BaseWalkSpeed * BaseRoleMult * SkillMult;
        }
    }

    // 기존 RoleType별 MaxHP, AttackDamage, KnockbackCoefficient 설정은 유지한다.
}

{
    if (!FrozenOverlayMeshComponent)
    {
        return;
    }

    if (FrozenOverlayMesh)
    {
        FrozenOverlayMeshComponent->SetStaticMesh(FrozenOverlayMesh);
    }

    if (FrozenOverlayMaterial)
    {
        FrozenOverlayMeshComponent->SetMaterial(0, FrozenOverlayMaterial);
    }

    FrozenOverlayMeshComponent->SetRelativeLocation(FrozenOverlayRelativeLocation);
    FrozenOverlayMeshComponent->SetRelativeScale3D(FrozenOverlayRelativeScale);
    FrozenOverlayMeshComponent->SetCollisionEnabled(ECollisionEnabled::NoCollision);
    FrozenOverlayMeshComponent->SetHiddenInGame(!bFrozen);
    FrozenOverlayMeshComponent->SetVisibility(bFrozen, true);
}

3-10. ApplyFreeze / EndFreeze 구현
--------------------------------

void AMyCharacter::ApplyFreeze()
{
    if (IsDead() || !CanReceiveStatusEffect(ERecStatusEffectType::Freeze))
    {
        return;
    }

    bFrozen = true;
    CachedMoveRight = 0.f;
    CachedMoveForward = 0.f;
    StopJumping();

    if (UCharacterMovementComponent* MoveComp = GetCharacterMovement())
    {
        MoveComp->StopMovementImmediately();
        MoveComp->MaxWalkSpeed = 0.f;
    }

    UpdateFrozenOverlay();
}

void AMyCharacter::EndFreeze()
{
    if (!bFrozen)
    {
        return;
    }

    bFrozen = false;

    UpdateFrozenOverlay();
    ApplyRoleStats();
}

void GameLogic::ApplyFreezeToPlayer(int sessionID, float duration)
{
    auto it = players.find(sessionID);
    if (it == players.end() || !it->second)
    {
        return;
    }

    Session* target = it->second;
    GameData& gd = target->gameDatas;

    if (!gd.isConnected || gd.characterState != 0)
    {
        return;
    }

    {
        std::lock_guard<std::mutex> lock(target->MoveIntentLock);
        target->bFrozen = true;
        target->FreezeRemainTime = duration;
        target->HorizontalVelocityX = 0.0f;
        target->HorizontalVelocityY = 0.0f;
    }

    BroadcastFreezeState(target->playerUID, 1, duration);
}

void GameLogic::EndFreezeOnPlayer(int sessionID)
{
    auto it = players.find(sessionID);
    if (it == players.end() || !it->second)
    {
        return;
    }

    Session* target = it->second;

    {
        std::lock_guard<std::mutex> lock(target->MoveIntentLock);
        if (!target->bFrozen)
        {
            return;
        }

        target->bFrozen = false;
        target->FreezeRemainTime = 0.0f;
    }

    BroadcastFreezeState(target->playerUID, 0, 0.f);
}

void GameLogic::BroadcastFreezeState(int targetUID, int bFrozen, float duration)
{
    if (!ownerRoom)
    {
        return;
    }

    FreezeStatePacket pkt{};
    pkt.targetID = targetUID;
    pkt.bFrozen = bFrozen;
    pkt.duration = duration;

    ownerRoom->BroadcastToMembers(
        PKT_S2C_FREEZE_STATE_BRD,
        reinterpret_cast<const char*>(&pkt),
        sizeof(pkt)
    );
}

    float BurnDuration = 2.f;

    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Torch|Thaw")
    float ThawRequiredTime = 1.5f;

    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Torch|Thaw")
    float ThawRangeBuffer = 0.f;

    UPROPERTY(VisibleInstanceOnly, BlueprintReadOnly, Category = "Torch|Thaw")
    TObjectPtr<AMyCharacter> CurrentThawTarget = nullptr;

    UPROPERTY(VisibleInstanceOnly, BlueprintReadOnly, Category = "Torch|Thaw")
    float ThawProgress = 0.f;

    TMap<AMyCharacter*, FTorchBurnState> ActiveBurns;

    void ApplyBurn(AMyCharacter* Target);
    void UpdateBurns(float DeltaTime);
    void TryStartThaw(AMyCharacter* Target);
    void UpdateThaw(float DeltaTime);
    void CancelThaw();
    bool IsValidThawTarget(AMyCharacter* Target) const;
    bool IsTargetInTorchRange(AMyCharacter* Target) const;

    UFUNCTION(BlueprintImplementableEvent, Category = "Torch")
    void BP_OnTorchHitTarget(AActor* Target);

    UFUNCTION(BlueprintImplementableEvent, Category = "Torch")
    void BP_OnUnfreezeTarget(AMyCharacter* Target);
};

TorchItem.cpp 핵심 구현
---------------------
#include "TorchItem.h"

#include "CollisionQueryParams.h"
#include "Engine/OverlapResult.h"
#include "Kismet/GameplayStatics.h"
#include "../../Sub/MyCharacter.h"

ATorchItem::ATorchItem()
{
    PrimaryActorTick.bCanEverTick = true;

    Damage = 0.f;
    KnockbackPower = 200.f;
    Range = 220.f;
    Radius = 110.f;
}

void ATorchItem::Tick(float DeltaTime)
{
    Super::Tick(DeltaTime);

    UpdateBurns(DeltaTime);
    UpdateThaw(DeltaTime);
}

void ATorchItem::DoHit()
{
    if (!OwnerCharacter || OwnerCharacter->IsDead() || !GetWorld())
    {
        CancelThaw();
        return;
    }

    const FVector Center =
        OwnerCharacter->GetActorLocation() +
        OwnerCharacter->GetActorForwardVector() * Range;

    TArray<FOverlapResult> Overlaps;
    FCollisionQueryParams Params;
    Params.AddIgnoredActor(OwnerCharacter);
    Params.AddIgnoredActor(this);

    const FCollisionShape Shape = FCollisionShape::MakeSphere(Radius);
    const bool bHit = GetWorld()->OverlapMultiByChannel(
        Overlaps,
        Center,
        FQuat::Identity,
        ECC_Pawn,
        Shape,
        Params
    );

    if (!bHit)
    {
        return;
    }

    for (const FOverlapResult& Result : Overlaps)
    {
        AMyCharacter* Target = Cast<AMyCharacter>(Result.GetActor());
        if (!Target || Target->IsDead())
        {
            continue;
        }

        ApplyBurn(Target);
        TryStartThaw(Target);
        BP_OnTorchHitTarget(Target);
    }
}

void ATorchItem::ApplyBurn(AMyCharacter* Target)
{
    if (!Target || Target->IsDead())
    {
        return;
    }

    FTorchBurnState& BurnState = ActiveBurns.FindOrAdd(Target);
    BurnState.RemainingTime = BurnDuration;
    if (BurnState.TimeUntilNextTick <= 0.f)
    {
        BurnState.TimeUntilNextTick = BurnTickInterval;
    }
}

void ATorchItem::UpdateBurns(float DeltaTime)
{
    if (DeltaTime <= 0.f || ActiveBurns.Num() == 0)
    {
        return;
    }

    for (auto It = ActiveBurns.CreateIterator(); It; ++It)
    {
        AMyCharacter* Target = It.Key();
        FTorchBurnState& BurnState = It.Value();

        if (!OwnerCharacter || OwnerCharacter->IsDead() || !Target || Target->IsDead())
        {
            It.RemoveCurrent();
            continue;
        }

        BurnState.TimeUntilNextTick -= DeltaTime;

        while (BurnState.RemainingTime > 0.f && BurnState.TimeUntilNextTick <= 0.f)
        {
            UGameplayStatics::ApplyDamage(
                Target,
                static_cast<float>(BurnDamagePerTick),
                OwnerCharacter->GetController(),
                this,
                nullptr
            );

            BurnState.TimeUntilNextTick += BurnTickInterval;
        }

        BurnState.RemainingTime -= DeltaTime;
        if (BurnState.RemainingTime <= 0.f)
        {
            It.RemoveCurrent();
        }
    }
}

void ATorchItem::TryStartThaw(AMyCharacter* Target)
{
    if (!IsValidThawTarget(Target))
    {
        return;
    }

    if (CurrentThawTarget != Target)
    {
        CurrentThawTarget = Target;
        ThawProgress = 0.f;
    }
}

void ATorchItem::UpdateThaw(float DeltaTime)
{
    if (!CurrentThawTarget)
    {
        return;
    }

    if (DeltaTime <= 0.f || !IsValidThawTarget(CurrentThawTarget))
    {
        CancelThaw();
        return;
    }

    ThawProgress += DeltaTime;
    if (ThawProgress >= ThawRequiredTime)
    {
        AMyCharacter* ThawedTarget = CurrentThawTarget;
        ThawedTarget->EndFreeze();
        BP_OnUnfreezeTarget(ThawedTarget);
        CancelThaw();
    }
}

void ATorchItem::CancelThaw()
{
    CurrentThawTarget = nullptr;
    ThawProgress = 0.f;
}

bool ATorchItem::IsValidThawTarget(AMyCharacter* Target) const
{
    return OwnerCharacter &&
        !OwnerCharacter->IsDead() &&
        Target &&
        !Target->IsDead() &&
        Target->IsFrozen() &&
        IsTargetInTorchRange(Target);
}

bool ATorchItem::IsTargetInTorchRange(AMyCharacter* Target) const
{
    if (!OwnerCharacter || !Target)
    {
        return false;
    }

    const FVector Center =
        OwnerCharacter->GetActorLocation() +
        OwnerCharacter->GetActorForwardVector() * Range;

    const float AllowedRange = Radius + ThawRangeBuffer;
    return FVector::DistSquared(Target->GetActorLocation(), Center) <= FMath::Square(AllowedRange);
}

중요한 수정 포인트
----------------
위 TorchItem 코드는 클라이언트 로컬 판정 예시다. 현재 프로젝트는 서버에서 GameLogic::HandleAttackHitReport를 통해 공격을 검증하므로 최종 구현은 다음처럼 바꿔야 한다.

1. 클라이언트 ATorchItem::DoHit()
   - 타겟 후보를 찾고 바로 ApplyDamage/EndFreeze를 호출하지 않는다.
   - 현재 프로젝트의 AMyCharacter::OnAttackOverlap처럼 UNetworkInstance::RequestAttackHitReport를 보내는 방식에 맞춘다.

2. 서버 GameLogic::HandleAttackHitReport(...)
   - 공격자의 gd.equippedItemID를 확인한다.
   - 해당 itemID가 횃불이면 일반 공격 피해 대신 BurnDamagePerTick / 해동 처리를 수행한다.
   - 대상이 bFrozen이면 EndFreezeOnPlayer(targetSession->sessionID)를 호출한다.

수정해야 할 부분
--------------
AMyCharacter::Attack() 안에서 현재 const int32 AttackType = CurrentItem ? 1 : 0;를 만들지만 RequestAttackInput(0)을 호출하고 있다.

현재 코드:

const int32 AttackType = CurrentItem ? 1 : 0;
NetworkInstance->RequestAttackInput(0);

수정 방향:

const int32 AttackType = CurrentItem ? 1 : 0;
NetworkInstance->RequestAttackInput(AttackType);

하지만 서버 GameLogic::HandleAttackInput(int sessionID)는 현재 const int attackType = 0;으로 고정되어 있다.

서버까지 제대로 하려면:
 - Packet.h의 AttackPacket을 실제로 읽도록 PacketProcessor::Handle_Attack_KeyInput 수정
 - GameLogic::HandleAttackInput(int sessionID, int attackType)로 인자 확장
 - BroadcastAttackAction(attackerUID, attackType, attackSeq)에 실제 attackType 전달

간단한 우선 구현:
 - attackType은 서버에서 gd.equippedItemID != -1이면 1, 아니면 0으로 결정한다.

GameLogic::HandleAttackInput 예시:

const int attackType = gd.equippedItemID != -1 ? 1 : 0;
